# Таможенный AI-помощник

Этот проект — Telegram-бот, предоставляющий справочную информацию по вопросам таможенного дела с использованием OpenRouter API и современных LLM-моделей.

## Возможности
- Ответы на вопросы по таможенному регулированию, процедурам, платежам, декларированию и ограничениям
- Форматирование ответов в виде отчёта, справки или письма
- Экспорт ответов в формате DOCX
- Асинхронная обработка запросов
- Логирование всех обращений

## Установка
1. Клонируйте репозиторий или скачайте файлы проекта.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл `.env` и добавьте переменные:
   ```env
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   OPENROUTER_API_KEY=ваш_ключ_openrouter
   OPENROUTER_MODEL=nex-agi/deepseek-v3.1-nex-n1:free
   OPENROUTER_BASE_URL=https://openrouter.ai/api/v1/chat/completions
   ```

## Запуск
```bash
python main.py
```

## Использование
- Отправьте текстовый вопрос по таможенному делу в чат с ботом.
- Доступные команды:
  - `/start` — приветствие
  - `/help` — справка
  - `/about` — информация о боте
  - `/status` — статус бота
  - `/style report|reference|letter` — смена стиля ответа
  - `/export docx` — экспорт последнего ответа в DOCX

## Пример вопроса
- Какие документы нужны для таможенного оформления?
- Как рассчитать таможенную пошлину?
- Какие товары запрещены к ввозу?

## Ограничения
- Ответы носят информационный характер и не являются юридической консультацией
- Возможны неточности, связанные с обновлением законодательства
